# 📝 Note Keeper App

A simple full-stack app to create and delete notes.

## 🛠 Tech Stack
- Frontend: React + Axios
- Backend: Node.js + Express
- Database: MongoDB
- Styling: Custom CSS

## 🚀 Features
- Create notes with title & content
- Delete notes
- Press Enter in title to jump to content
- Clean, responsive UI

## 📦 How to Run

### Backend
```bash
cd backend
npm install
npm start
